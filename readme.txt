== Description ==
WebSec will automatically check your website for potential security vulnerabilities. You are able to see instantly where you could improve your security by just adjusting some basic settings.

You are responsible for the security of your website and all data behind. So it's up to you. We will make our security expertise easily understandable and available for you. Just install the plugin and see your security status. 

WebSec is free of charge. We will change no settings of your website without asking - for sure.
